# Header

| Offset | Size | Description |
| ------ | ---- | ----------- |
| 0x00   | 4    | Magic ("RARC") |
| 0x04   | 4    | Total filesize |
| 0x08   | 4    | Header size    |
| 0x0C   | 4    | Data offset (add header size) |
| 0x10   | 4    | Data size |
| 0x14   | 4    | 0 |
| 0x18   | 4    | Data size |
| 0x1C   | 4    | 0 |
